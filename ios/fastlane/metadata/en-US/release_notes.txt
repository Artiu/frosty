This update contains various chat enhancements and improvements in layouts and styles.

New Features
• Added replied message to replies in chat
• Added first message highlights
• Added options to hide highlights and notices in chat
• Added an option to move the emote menu button to the left side

Improvements
• Moved settings panel to the "More" button (three dots) in chat
• Changed arrow icon for resume scroll button
• Reduced title opacity and font-weight on stream cards for consistency
• Renamed the "Followed Streams" tab to just "Following"
• Darkened the dark theme (again)

Bug Fixes
• Fixed misaligned alert messages
• Fixed misaligned chat names when setting message scale above 1.0
